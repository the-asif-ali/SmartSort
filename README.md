📂 SMART SORT – FILE ORGANIZER
👨‍💻 Author: Asif Ali

════════════════════════════════════════════════════

📌 ABOUT THE PROJECT
Smart Sort is a simple and useful Python program that 
automatically organizes files in a folder into categories 
like Images, Videos, Documents, etc.

✨ It helps you clean messy folders quickly and easily.

════════════════════════════════════════════════════

🚀 FEATURES
✔ Automatically sorts files based on file extension
✔ Creates folders like Images, Videos, Audio, Documents, etc.
✔ Skips duplicate files (no overwriting)
✔ Beginner-friendly and easy to use

════════════════════════════════════════════════════

🛠 REQUIREMENTS
✔ Python 3.x installed
✔ No external libraries needed

════════════════════════════════════════════════════

📦 MODULES USED
📁 os       → Handles files and folders
📦 shutil   → Moves files

════════════════════════════════════════════════════

▶️ HOW TO USE

🔹 Step 1:
Run the program:
    python smart_sort.py

🔹 Step 2:
Enter the folder path when asked

💡 Example:
    Enter the destination folder :
    C:\Users\Asif\Downloads

════════════════════════════════════════════════════

⚙️ HOW IT WORKS

1️⃣ Reads all files in the folder  
2️⃣ Checks file extensions  
3️⃣ Decides category (Images, Videos, etc.)  
4️⃣ Creates folders if needed  
5️⃣ Moves files into correct folders  

════════════════════════════════════════════════════

📂 FILE CATEGORIES

🖼 Images        → .jpg, .jpeg, .png, .gif, .bmp, .webp, .svg  
🎬 Videos        → .mp4, .mkv, .avi, .mov, .flv, .wmv  
🎵 Audio         → .mp3, .wav, .aac, .flac, .ogg  
📄 Documents     → .txt, .pdf, .doc, .docx, .xls, .xlsx, .ppt, .pptx  
🗜 Archives      → .zip, .rar, .7z, .tar, .gz  
💻 PC Apps       → .exe, .msi, .bat, .cmd  
📱 Mobile Apps   → .apk, .xapk, .ipa  
🧑‍💻 Code Files    → .py, .java, .c, .cpp, .js, .html, .css, .dart, .php  
⚙️ System Files  → .dll, .sys, .ini  
🔤 Fonts         → .ttf, .otf, .woff  
💿 Disk Images   → .iso, .img  
❓ Others        → Unknown file types  

════════════════════════════════════════════════════

⚠️ IMPORTANT NOTES

🔸 Only main folder files are organized  
🔸 Subfolders are NOT affected  
🔸 Duplicate files are skipped  
🔸 Enter correct folder path  

════════════════════════════════════════════════════

📊 EXAMPLE

📥 Before:
Downloads/
    file1.jpg
    song.mp3
    notes.pdf

📤 After:
Downloads/
    Images/file1.jpg
    Audio/song.mp3
    Documents/notes.pdf
